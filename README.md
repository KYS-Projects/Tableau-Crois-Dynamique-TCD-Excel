# Tableau-Crois-Dynamique-TCD-Excel
L’objectif de ce projet est de travailler avec une base de données afin de réaliser différentes analyses à l’aide des tableaux croisés dynamiques (TCD) dans Excel.
La base de données utilisée est fictive et représente une entreprise composée de 284 employés, répartis sur quatre sites différents.
Les âges des employés varient de 20 à 65 ans, et sont regroupés par tranches d’âge de cinq ans afin de faciliter les analyses.
À partir de cette base de données, plusieurs analyses sont réalisées en utilisant l’outil Rapport de tableaux croisés dynamiques. Le projet consiste à résoudre huit exercices permettant d’explorer et de synthétiser les informations contenues dans les données.
