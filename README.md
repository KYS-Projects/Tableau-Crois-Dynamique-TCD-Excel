# Tableau-Crois-Dynamique-TCD-Excel
Analyse d’une base de données fictive représentant une entreprise de 284 employés répartis sur 4 sites. Les âges varient de 20 à 65 ans (tranches de 5 ans). L’objectif est de réaliser 8 analyses avec les tableaux croisés dynamiques (TCD) dans Excel afin d’explorer et synthétiser les données.
